# MILK — Products & Services

## Overview

Milk delivers AI-powered execution systems through a structured engagement model. This document defines what Milk offers, how engagements work, and what's in and out of scope.

---

## Service Model

Milk is not a product you buy off the shelf. It's an engagement — a partnership where we audit your workflows, build the execution layer, and optionally run it for you.

### The Three Phases

#### Phase 1: Audit
**Duration:** 1–2 weeks
**What happens:**
- Deep-dive into current workflows across target functions
- Map where time is being wasted, where bottlenecks exist, where work is repetitive
- Identify high-impact workflows for AI-powered execution
- Deliver an Execution Blueprint: a document showing which workflows will be systemized, how, and expected impact

**Deliverable:** Execution Blueprint (workflow maps, prioritized automation opportunities, recommended agent architecture)

#### Phase 2: Build
**Duration:** 2–4 weeks
**What happens:**
- Design workflow logic and agent orchestration
- Build agents with appropriate tools, prompts, and decision logic
- Integrate with existing tools (Slack, WhatsApp, CRM, spreadsheets, dashboards, email)
- Set up interfaces (chat, dashboard, or embedded in existing tools)
- Internal testing and QA
- Team onboarding and training

**Deliverable:** Running execution system with documentation, trained team, and support channel

#### Phase 3: Run (Optional)
**Duration:** Ongoing (monthly retainer)
**What happens:**
- Milk operates and monitors the execution layer
- Continuous optimization based on output quality and team feedback
- Add new workflows as needs evolve
- Handle edge cases, exceptions, and system updates
- Monthly performance reports

**Deliverable:** Managed execution with monthly reporting and optimization

## Engagement Tiers

### Tier 1: Pilot / Design Partner
**Best for:** First clients, validation engagements
**Scope:** 1–2 workflows, single function (e.g., marketing OR sales)
**Duration:** 4–6 weeks (Audit + Build)
**Includes:**
- Workflow audit for one function
- 1–2 agent workflows built and deployed
- Integration with up to 3 tools
- 2 weeks of post-launch support
- Monthly check-in for 2 months

**Pricing:** Reduced rate in exchange for feedback, iteration access, and case study rights

### Tier 2: Function-Level Execution
**Best for:** Companies ready to systemize an entire function
**Scope:** Full function coverage (e.g., all marketing workflows, or all sales workflows)
**Duration:** 6–8 weeks (Audit + Build) + optional Run
**Includes:**
- Comprehensive workflow audit for one function
- 3–6 agent workflows built and deployed
- Integration with up to 6 tools
- Team training and documentation
- 4 weeks of post-launch support
- Optional managed execution (monthly retainer)

### Tier 3: Enterprise Execution Layer
**Best for:** Companies wanting cross-functional AI execution
**Scope:** Multiple functions, organization-wide
**Duration:** 8–12 weeks (Audit + Build) + Run
**Includes:**
- Cross-functional workflow audit
- 6–15+ agent workflows across functions
- Full integration suite
- Custom dashboards and reporting
- Dedicated support channel
- Managed execution with monthly optimization

## What's In Scope

- Workflow design and mapping
- Agent creation, prompting, and orchestration
- Tool integration (APIs, webhooks, existing platforms)
- Chat interface setup (Slack, WhatsApp, web)
- Team training and onboarding
- Documentation and runbooks
- Post-launch support and bug fixes
- Ongoing optimization (in managed execution)

## What's Out of Scope

- Building custom software applications (we build execution systems, not apps)
- Data infrastructure or warehouse setup
- Hardware procurement or IT infrastructure
- Replacing existing CRMs, ERPs, or core business systems (we integrate with them)
- One-off consulting or strategy-only engagements (we build, not just advise)
- Content creation without a workflow system (we're not a content agency)

## Key Metrics We Track

For every engagement, Milk tracks and reports on:
- **Time saved** — Hours per week reclaimed from manual work
- **Output volume** — Quantity of outputs generated (proposals, reports, campaigns, etc.)
- **Speed improvement** — Time from request to deliverable (before vs. after)
- **Quality scores** — Client satisfaction with agent outputs (1–5 scale)
- **Adoption rate** — Percentage of team actively using the system
- **Cost efficiency** — Cost per output vs. previous manual cost

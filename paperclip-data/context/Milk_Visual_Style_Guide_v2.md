**Version 2.0 · March 2026 · Confidential**

## 1. Brand Overview

### Brand Essence

Milk is an execution enablement partner that builds and operates AI-powered workflow systems. The visual identity communicates: bold confidence, systematic precision, and a disruptive edge. Milk looks like a company that ships, not one that theorizes.

### Design Philosophy

**Bold, not loud:** Every visual element should command attention through confidence, not noise. Strong typography, decisive color, generous whitespace.

**Systematic, not sterile:** The design should feel engineered and intentional, but never cold. There's warmth in the animal characters and the hand-lettered logo.

**Disruptive, not gimmicky:** Milk's visual identity breaks conventions on purpose — animal-headed business characters, a hand-drawn wordmark in a tech space. These are brand choices, not accidents.

### Brand Personality in Visuals

| Trait | Visual Expression |
|-------|-------------------|
| Smart | Clean layouts, clear hierarchy, data-informed design decisions |
| Sharp | Precise spacing, deliberate typography, no visual clutter |
| Bold | High-contrast color palette, oversized headings, confident statements |
| Disruptive | Animal characters in business attire, hand-lettered logo, unconventional imagery |

## 3. Color Palette

### Brand Colors

Milk's color system is built around four core colors. The palette is bold and high-contrast, anchored by the signature Milk Red and balanced with a near-black, a clean yellow accent, and a warm light gray.

| Swatch | Name | Hex | RGB | Usage |
|--------|------|-----|-----|-------|
| 🔴 | **Milk Red** | #DC2625 | 220, 38, 37 | Primary CTA, logo on light backgrounds, key accents, links, emphasis text |
| ⬛ | **Milk Dark** | #2B2B2B | 43, 43, 43 | Primary text, headings, dark backgrounds, navigation |
| 🟡 | **Milk Yellow** | #F0F37E | 240, 243, 126 | Highlight accents, underlines, badges, tags, background blocks for character cards |
| ⬜ | **Milk Light** | #F4F4F4 | 244, 244, 244 | Page backgrounds, section dividers, cards |

> ⚠️ **IMPORTANT: Only Milk Yellow #F0F37E is approved.** Do not use any other yellow variant. No gold, amber, lemon, or warm yellow. #F0F37E everywhere — backgrounds, accents, badges, cards, all applications.

### Extended Palette

| Swatch | Name | Hex | RGB | Usage |
|--------|------|-----|-----|-------|
| 🔴 | Logo Red | #F60001 | 246, 0, 1 | Logo on dark backgrounds only (brighter variant) |
| ⬜ | White | #FFFFFF | 255, 255, 255 | Backgrounds, cards, text on dark surfaces |
| ⬛ | Black | #000000 | 0, 0, 0 | Deep backgrounds, maximum contrast contexts |
| 🟤 | Milk Beige | #F1EFE0 | 241, 239, 224 | Logo variant, soft contrast text on dark backgrounds |

### Color Ratios

The brand should follow an approximate color distribution:

- **60% — White / Milk Light:** Backgrounds and breathing room. Keeps Milk from feeling loud.
- **25% — Milk Dark:** Text, headings, navigation. The workhorse color.
- **10% — Milk Red:** CTAs, logo, links, key accent moments. Used decisively, never decoratively.
- **5% — Milk Yellow (#F0F37E):** Highlight underlines, small badges, selective emphasis, character card backgrounds. The surprise element.

### Color Accessibility

- Milk Red (#DC2625) on white (#FFFFFF): ~5.3:1 contrast ratio — meets WCAG AA for normal text
- Milk Dark (#2B2B2B) on white: ~12.7:1 — exceeds WCAG AAA
- Always test red text at small sizes; use Milk Dark for body text, reserve red for larger elements and buttons

## 4. Typography

### Primary Font: Inter

Inter is Milk's sole typeface across all applications — digital, print, social, collateral. It is clean, highly readable, and available in the full weight range needed for Milk's typographic hierarchy.

**Download:** [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)

### Weight System

| Weight | Usage | Notes |
|--------|-------|-------|
| **ExtraBold (800)** | Hero headlines, large display text | Primary impact weight. Used for the biggest, boldest statements. |
| **Bold (700)** | Section headlines, sub-headlines, card titles | Standard heading weight across web and collateral. |
| **Medium (500)** | UI elements, buttons, labels, navigation, tags | Interactive and functional elements. |
| **Regular (400)** | Body text, descriptions, captions, long-form content | Default reading weight. |

### Typographic Scale (Web)

| Element | Size | Weight | Color | Line Height | Notes |
|---------|------|--------|-------|-------------|-------|
| Hero Headline | 48–64px | ExtraBold (800) | Milk Dark #2B2B2B | 1.1 | Tight letter-spacing. Split-color with Milk Red for emphasis words. |
| Hero Accent Text | 48–64px | ExtraBold (800) | Milk Red #DC2625 | 1.1 | The emphasis portion of split-color headlines. |
| Section Headlines | 32–40px | Bold (700) | Milk Dark | 1.2 | — |
| Sub-headlines | 20–24px | Bold (700) | Milk Dark | 1.3 | — |
| Body Text | 16–18px | Regular (400) | Milk Dark | 1.5–1.6 | — |
| Buttons & UI | 14–16px | Medium (500) | White (on red) or Milk Dark | 1.0 | Pill-shaped buttons, navigation items. |
| Eyebrow / Label | 12–14px | Medium (500) | Milk Red | 1.4 | Uppercase, letter-spacing 0.05–0.1em. |
| Tags / Badges | 12–13px | Medium (500) | Milk Dark | 1.0 | Outlined pills, light gray background. |
| Caption / Footer | 12–14px | Regular (400) | #888888 | 1.4 | — |

### Typographic Patterns

**Eyebrow + Headline:** Red uppercase eyebrow label above the main headline. Example: "UNLOCK MORE FROM YOUR TEAM" (red, small) above "Your team does the thinking. Agents do the rest." (dark, large). Use consistently across sections.

**Split-color headlines:** Headlines use Milk Dark for the first part and Milk Red for the emphasis phrase. This is a signature pattern. Example: "We don't sell tools." (dark) + "We set up the system." (red).

**Yellow underline highlights (#F0F37E):** A yellow underline accent on select key phrases. Use sparingly — maximum one per section. The underline should feel hand-placed, not systematic.

### Typography Rules

- Only Inter. No secondary typefaces. The hand-lettered logo is the only non-Inter element.
- Never use Inter Light or Thin — the minimum weight is Regular (400).
- Headings are always Milk Dark or Milk Red. Never gray, never other colors.
- Body text is always Milk Dark. Never red for body copy.
- Button text is always Medium (500) weight — not Bold, not Regular.

## 5. Imagery & Art Direction

### Animal Character System

Milk's most distinctive visual element is its cast of animal-headed characters in business attire. This is a deliberate brand choice: serious about business, but doesn't take itself too seriously.

### Character Roster

| Character | Function | Personality | Card Accent |
|-----------|----------|-------------|-------------|
| Tiger | Founder / Leadership | Bold, decisive, strategic | Red bg |
| Zebra | Marketing | Creative, distinctive, stands out | Red bg |
| Wolf | Sales | Sharp, persistent, outcome-focused | Yellow #F0F37E bg |
| Horse | Operations | Methodical, dependable, organized | Yellow #F0F37E bg |
| Bull | Finance | Strong, reliable, results-driven | Yellow #F0F37E bg |
| Elephant | HR | Thorough, never forgets | Red bg |

### Character Card Design

The character cards on the website follow a specific structure:
- **Milk Red (#DC2625) or Milk Yellow (#F0F37E) background block** behind the upper portion of the character (Tiger, Zebra, Elephant use Red; Wolf, Horse, Bull use Yellow)
- Character in **grayscale (B&W)** — business attire, professional poses
- **White card area** below with: function title (bold), subtitle (regular, gray), "WHAT THEY ASK MILK:" label (uppercase, yellow dot + gray text), sample prompt in *italic*, and output tags as outlined pills

### Full Character Card Set

**Tiger = Founder/Leadership:** Red bg. Prompt: "Give me a 5-min brief on what matters this week." Outputs: expense summary, key insights, action list.

**Zebra = Marketing:** Red bg. Prompt: "Build a campaign route for our launch by 4 PM." Outputs: ad concepts, captions, landing page, report summary.

**Wolf = Sales:** Yellow (#F0F37E) bg. Prompt: "Draft a proposal and follow-up before the call." Outputs: proposal draft, email, talking points.

**Horse = Operations:** Yellow (#F0F37E) bg. Prompt: "Turn this messy meeting into tasks, owners, and deadlines." Outputs: task list, owners, timeline, summary.

**Bull = Finance:** Yellow (#F0F37E) bg. Prompt: "Forecast spending and savings for the upcoming quarter." Outputs: spreadsheets, graphs, reports.

**Elephant = HR:** Red bg. Prompt: "Set up onboarding for 3 new joiners next week." Outputs: checklist, welcome flow, policy answers.

### Photography Style

**Character photos:** Black and white photography with animal heads composited. Business-casual to business-formal attire. Confident postures. Clean studio backgrounds.

**Color treatment:** Character images are grayscale. Color accents come from Milk Yellow (#F0F37E) background blocks and Milk Red UI elements — never colorized into the photography itself.

**No stock photography:** All imagery either features the animal characters or shows actual product interfaces.

## 6. Social Media Style Reference

### Instagram Grid Aesthetic

The Instagram grid follows a deliberately varied but cohesive visual system. Based on the approved style reference:

### Post Style Categories

**Type 1 — Dark Background, Bold Typography**
- Black or near-black (#000000 / #2B2B2B) background
- Large white or Milk Red text — Inter ExtraBold
- Minimal imagery, text-dominant
- Example: "you say what needs to be done. the system runs it."

**Type 2 — Yellow Background, Mixed Media**
- Milk Yellow (#F0F37E) background blocks
- Layered composition: screenshots, UI elements, red/black accents
- Energetic, dynamic, slightly chaotic-on-purpose
- Example: "you say what needs to be. the system runs it." with floating UI mockups

**Type 3 — Animal Character Features**
- Character prominently placed with background color accents
- Can use yellow, red, or dark backgrounds
- Often includes text overlay with the brand message
- Example: Animal characters in suits with "Your team focuses on decisions. The system runs everything else."

**Type 4 — Product/System Visualization**
- Shows Milk's system in action: dashboards, chat interfaces, workflow diagrams
- Dark background with glowing/neon accents (red, yellow-green)
- Tech-forward but not cold — the Milk system feels alive
- Example: Phone mockup showing Milk chat interface with "Run quarterly marketing report"

**Type 5 — White/Light Background, Statement**
- Clean white or light gray background
- Bold black headline with selective red or yellow accent
- Minimal, confident, typography-led
- Example: "Your team has ideas. Milk runs the work." with small logo

### Grid Rhythm

- Alternate between dark and light posts — never 3 dark posts in a row
- Mix text-dominant with image-dominant — variety in visual density
- Animal characters appear every 3–4 posts to maintain brand recognition
- Yellow (#F0F37E) accents appear consistently but not in every post

### Instagram Visual Rules

1. Every post uses **only** the approved palette: #2B2B2B, #DC2625, #F0F37E, #F4F4F4, #FFFFFF, #000000
2. Typography is always Inter — ExtraBold for headlines, Medium for UI elements
3. Lowercase is the default for headlines on social. Sentence case, not title case. ("one system runs it all." not "One System Runs It All.")
4. Animal characters are always in grayscale with color only in the background/accents
5. UI/product mockups should look real and functional, not wireframe-style

## 11. Do's and Don'ts

### Do
- Use generous whitespace — confidence comes from restraint
- Lead with typography — headlines should be large, bold, and carry the message
- Use the animal characters consistently — maintain the character-function mapping
- Show the product — embed real demos and screenshots wherever possible
- Use split-color headlines — dark + red is a brand signature
- Use yellow (#F0F37E) underlines sparingly — max one per section
- Use lowercase for social media headlines — it's more confident than title case
- Vary post styles — the grid should feel alive, not templated

### Don't
- Don't use gradients
- Don't use stock photography
- Don't overuse red — it's for CTAs, logo, and emphasis, not decoration
- Don't add decorative elements (swooshes, starbursts, drop shadows, decorative borders)
- Don't use more than Inter + the logo — no secondary typefaces
- Don't mix character meanings (tiger is always leadership, wolf is always sales, etc.)
- Don't make it corporate — if it could be any SaaS company, it's not Milk
- Don't use any yellow other than #F0F37E — no gold, amber, lemon, or warm yellows
- Don't copy-paste the same layout across posts — frameworks, not templates

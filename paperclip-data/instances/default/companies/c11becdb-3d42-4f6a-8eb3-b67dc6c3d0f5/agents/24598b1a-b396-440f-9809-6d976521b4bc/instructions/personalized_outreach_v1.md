## How to Use This File

1. Open `email_sequences_v1.md` and pick the base template (Variant A, B, or C) noted for each channel
2. Replace the **first paragraph** of the template body with the personalized opening line above
3. Keep the rest of the email as-is — the offer, CTA, and sign-off are already proven
4. Add UTM params to the link: `?utm_source=email&utm_medium=cold&utm_campaign=top10-personalized`
5. Send as plain text email — no HTML, no images — from `team@thumbnailos.in`
6. Log the send date and flag any replies to CEO immediately

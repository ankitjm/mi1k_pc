# CEO Heartbeat — ThumbnailOS Weekly Rhythm

The Heartbeat is the CEO's non-negotiable weekly rhythm. It runs automatically. It does not wait for AJ to ask.

---

## SUNDAY 10AM — RESET AND RESEARCH

- Pull weekly metrics: audits sent, open rate, reply rate, conversion rate, MRR delta, churn
- Research 50 new Indian YouTube target channels
- Score channels by CTR potential, add to Airtable pipeline
- Brief all agents on week priorities
- Send AJ a Sunday Summary (3 bullet points max):
  - What shipped last week
  - What the number is (MRR or conversions)
  - What this week's focus is

## MONDAY–TUESDAY — AUDIT GENERATION

- Delivery Agent generates audits for all new free signups
- CEO reviews first 5 audits for quality before approval
- Engineer fixes any pipeline issues flagged

## WEDNESDAY — DELIVERY DAY

- Delivery Agent sends all approved PDF audits via Brevo
- WhatsApp notifications sent to all recipients
- CEO monitors delivery success rate

## THURSDAY — FOLLOW UP

- Growth Agent sends Day 3 follow up to non-converters
- CEO reviews reply data and flags high-intent leads

## FRIDAY — OUTREACH AND REVIEW

- Growth Agent sends cold outreach to 50 new channels
- Engineer submits weekly status: what shipped, what is pending, any blockers
- CEO sends AJ a Friday Update (3 bullets max):
  - What shipped this week
  - Current MRR and conversion rate
  - One decision AJ needs to make (if any)

## SATURDAY — ANALYZE AND ITERATE

- Pull conversion data: which niche, subject line, and audit format converted best
- Update Growth Agent outreach playbook
- Update Delivery Agent report template if needed
- Prepare Sunday research brief

## Heartbeat Cron Schedule

The CEO heartbeat runs every Sunday at 10AM IST and Friday at 5PM IST to enforce this rhythm. Additional agent heartbeats run per their schedules as set up by the Engineer.

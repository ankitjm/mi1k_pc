## How to Use This Log

1. **Opening stock** — count all items before service starts, record qty.
2. **Closing stock** — count remaining stock after service, record qty.
3. **Used today** = Opening Qty − Closing Qty (minus any deliveries received).
4. **Wastage** = anything discarded (burnt dough, spoiled veg, dropped items). Always note the reason.
5. **Reorder** — if closing stock is at or below the reorder point, tick and raise a purchase request with the supervisor.
6. **Food cost %** = (Estimated food cost ÷ Revenue) × 100. Target: ≤ 30%.

> **Rule:** Never leave the stall without completing both sections. Unsigned logs will not be accepted.

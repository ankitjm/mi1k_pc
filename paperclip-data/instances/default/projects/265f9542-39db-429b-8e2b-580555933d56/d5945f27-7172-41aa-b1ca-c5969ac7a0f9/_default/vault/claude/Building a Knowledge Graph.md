# Building a Knowledge Graph

## User  
*2024-04-01 10:00:00*

How can I build a personal knowledge graph from my chat history?

## Assistant  
*2024-04-01 10:01:00*

You can parse chat exports into structured Markdown with YAML frontmatter, then use Obsidian's graph view to visualize connections. Tag conversations by topic and use wikilinks to create cross-references between related conversations.

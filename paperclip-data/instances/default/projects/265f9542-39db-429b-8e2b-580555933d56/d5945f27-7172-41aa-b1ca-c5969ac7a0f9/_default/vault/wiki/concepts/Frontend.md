# Frontend — React

## Component Architecture

Ankit has explored React component structure for large applications:

- **Atomic design pattern**: atoms → molecules → organisms → templates → pages
- Single responsibility principle: each component does one thing
- Keep components small and focused; composition over large monoliths

(source: [[chatgpt/React Component Architecture]])

## Key Themes

- Applies software design principles (SRP, composition) to UI architecture
- Building larger applications, not just prototypes — interested in scalable patterns
